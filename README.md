## About LINQ  

* C#3.0で登場した、リレーショナルデータベースやXMLに対する動作をプログラミング言語に統合するもの。
* Language INtegrated Queryの略
* 様々なタイプのデータソースに対する処理を簡単・簡潔に行える
* C#でSQLライクなデータベース操作構文を組み込むライブラリ
* [福岡のアイドルグループ](http://loveinq.com/)ではない
